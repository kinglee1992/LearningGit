# LearningGit



- ## 版本库（仓库，repository）

  命令：

  - 创建仓库  git init

    注意事项：

    1. 只能管理文本文件；推荐使用UTF-8编码；
    2. Windows下不要使用记事本编辑文件；推荐使用Notepad++，编码选择UTF-8 without BOM;

  - 把文件添加到仓库  git add readme.txt

  - 把文件提交到仓库  git  commit -m  "INI: this is a initial commit"
